# recursive-hyperoperation-logic

Algorithm that enabled 100% IRR for 12 consecutive years.

## 本リポジトリの位置づけ

本リポジトリは、生み出してきた理論の**総まとめ**であり、脳のバックグラウンドプロセスで動いている自動プログラムを**ハードコード**し、脳の外部ストレージに移植することで、脳を開け、今後12年も同様のIRRパフォーマンスを達成できるようにするための**外部記録媒体**作成の試みである。

（HST: 人間はストレージが外部化されたコンピューティングデバイス。永続的な情報は境界面の外側に置かれる。）

## 成長の位相：指数関数ではなく Hyperoperation

本成果は**指数関数的成長**（a^x）ではなく、**tetration 級・hyperoperation** に近い。

| 演算階層 | 記法 | 例 | 経営における対応 |
|---------|------|-----|----------------|
| H0 加法 | a + b | 1 + 1 | 単年利益の足し算 |
| H1 乗法 | a × b | 2 × 3 | 売上×利益率 |
| H2 冪乗 | a^b | 2^10 | 複利、指数関数的成長 |
| H3 テトレーション | a↑↑b | 2↑↑3 = 2^2^2 | 再帰的複利、構造の自己相似 |
| H4+ | a↑↑↑b... | — | hyperoperation |

**centillion**（10^303）という大数を扱えるから、この成果が出せる。指数関数の枠組みでは到達できない領域に、hyperoperation の再帰によって到達する。

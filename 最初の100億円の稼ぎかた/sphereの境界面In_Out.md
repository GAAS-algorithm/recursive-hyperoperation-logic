## 2. 別の数理：本の見た目をしたブラックホールの境界面

理論上、**本の見た目をしたブラックホールの境界面**を作るとしたら、世界中の **y=2x になるオポチュニティが手元に集まってくる**はずである。

### 従来型との対比

| | 従来型 | 本の数理 |
|--|--------|----------|
| **本の役割** | 知識の容器。読者が取り出して使う | ブラックホールの境界面。幾何学的に存在する |
| **機会の流れ** | 読者が学び、探し、掴む（能動的探索） | 境界面の幾何学が時空を歪め、機会がこちらへ集まる（引力） |
| **因果** | 本 → 知識 → 行動 → 機会（因果鎖） | 境界面の存在 → オポチュニティ空間の曲率 → 収束（重力） |

### 境界面の In/Out（引力モデル）

| | 内容 |
|--|------|
| **In** | 境界面の存在そのもの（その幾何学的な正しさ） |
| **Out** | 世界中の y=2x オポチュニティが手元に収束すること |

### 含意

- 本は**能動的な引力源**である
- 境界面の設計が正しければ、機会は**探索ではなく収束**で手元に来る
- Functor は「知識の写像」ではなく、「オポチュニティ空間の曲率を規定する変換」になる
- 形式化には、オポチュニティ空間の計量、境界面の曲率、y=2x の「流線」の収束を定義する必要がある
